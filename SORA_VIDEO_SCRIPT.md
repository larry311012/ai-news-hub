## Audio Guidelines

### Background Music:
- **Genre:** Modern tech/corporate
- **Mood:** Uplifting, productive, professional
- **Tempo:** Medium (110-120 BPM)
- **Volume:** Subtle, doesn't overpower voiceover
- **Recommendation:** Royalty-free from Epidemic Sound or Artlist

### Voiceover:
- **Voice:** Professional, friendly, energetic
- **Pace:** Conversational (not too fast)
- **Tone:** Helpful, confident
- **Gender:** Neutral or your preference
- **Alternative:** Use AI voice (ElevenLabs, Play.ht) for consistency

### Sound Effects:
- Subtle "whoosh" for transitions
- Soft "ding" for checkmarks
- Gentle "pop" for UI elements appearing
- Keep minimal and professional

---

## Text Overlay Guidelines

### Font:
- **Primary:** Inter or SF Pro Display (clean, modern)
- **Size:** Large enough for mobile viewing
- **Color:** White text with dark semi-transparent background for readability
- **Animation:** Fade in/out, not distracting

### Placement:
- Lower third for most text
- Center for emphasis (like "AI News Hub" logo)
- Ensure text doesn't cover important UI elements

## Production Notes

### If Using Sora:
1. Generate each scene separately for better control
2. Use consistent lighting and camera angles
3. Request "professional software demo aesthetic" in prompts
4. Specify "smooth transitions" between cuts
5. Export at 1080p minimum

### Post-Production:
1. Add background music (subtle, professional)
2. Add voiceover or captions
3. Color grade for consistency (slightly desaturated, professional look)
4. Add text overlays for key points
5. Ensure smooth transitions between Sora-generated scenes
6. Add final end card with GitHub link

### Accessibility:
- Include closed captions
- Ensure text is readable on mobile
- Use high contrast for important text
- Don't rely solely on color to convey information
